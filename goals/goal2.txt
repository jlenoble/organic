Have Gulp write hello world.

Install gulp:
> npm i --save-dev git://github.com/gulpjs/gulp.git#4.0"

Write gulpfile.js:
const gulp = require('gulp');

gulp.task('default', () => {
  console.log('hello world');
  return Promise.resolve();
});

(Current node version is 8.5)

> gulp
[16:25:36] Using gulpfile ~/Projets/organic/gulpfile.js
[16:25:36] Starting 'default'...
hello world
[16:25:36] Finished 'default' after 3.07 ms

Goal is reached.
